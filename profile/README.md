## What is CCHub?

CCHub.ai is an independent bilingual community for Chinese-speaking Claude Code developers. We do four things:

- 📰 **Curated bilingual newsletter** — best of Claude in EN/中, weekly
- ❓ **Peer Q&A with SLA** — 2-hour response from verified answerers
- 🚀 **Showcases** — real Claude-built projects with full Build/Ship breakdowns
- 🏆 **Reputation system** — 段位 ranks accruing across contributions
